Send custom email message to your WooCommerce customers directly from WC Orders Admin Page.

== Description ==

Send custom email message to your WooCommerce customers directly from WC Orders Admin Page.
Select a list of orders, compose your message and send it via email to all selected customers.

The custom email messages have the same header and footer of all your WooCommerce email messages.

You can send a message to customers who bought a certain product. Try using our plugin [Search order by product SKU for WooCommerce](https://wordpress.org/plugins/search-order-by-product-sku-for-woocommerce/) to select orders that includes that product.

== Installation ==

Steps to install this plugin.

Be sure you're running WooCommerce 2.0+

1. In the downloaded zip file, there is a folder with name 'woo-send-email'
1. Upload the 'woo-send-email' folder to the '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Read the usage instructions in the description.
